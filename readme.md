## tools

Install tools... 

```sh
npm init @eslint/config
./node_modules/.bin/eslint yourfile.js
```

```sh
npm install --save-dev nodemon
npm i -D nodemon
```

...and write scripts in `package.json`

```json
"scripts": {
    "dev": "nodemon index.js",
    "lint": "./node_modules/.bin/eslint index.js"
    ...
}
```
